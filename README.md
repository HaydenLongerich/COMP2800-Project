A fully build Student Advisory System for the University of Windsor, Built as our COMP2800 Group Project.

This application allows students to browse courses, plan semesters, manage enrollments, and see their schedules through a calendar display.

**Includes:**
Home Page
Course Catalog
Enrollment System
Calendar Display
Advisor system
Reports System
Full Fledged Admin System

**Tech Stack:**
Frontend: JavaFX
Backend: Spring Boot (Java 21)
Database: MariaDB
Build Tool: Maven
Github

**SETUP INSTRUCTIONS:**
**IMPORTANT: Make sure to be running the University of Windsor Global Protect VPN**

git clone https://github.com/HaydenLongerich/COMP2800-Project.git
cd COMP2800-Project

Environment Variables:
$env:DB_URL=""
$env:DB_USERNAME=""
$env:DB_PASSWORD=""

Running the Application:
./mvnw javafx:run
